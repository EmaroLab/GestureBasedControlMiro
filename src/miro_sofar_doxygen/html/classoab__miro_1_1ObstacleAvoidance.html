Callback that receives the data from the robot sensors, and uses the information given by the sonar sensor to evaluate the presence of an obstacle.


If an obstacle is detected then is used the information related to the user's comand to start turning in one direction of few degree.
Each time the same obstacle is detected, the degrees respect to wich turns increases of a fixed quantity.
This angular velocity and a red lightening pattern for Miro's body are used to construct the platform_control message to publish
